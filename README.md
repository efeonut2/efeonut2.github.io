# Mehmet Önüt Emlak Website

Bu proje, Mehmet Önüt Emlak için hazırlanmış premium, statik bir gayrimenkul web sitesidir.

## Özellikler
- **Responsive Tasarım**: Mobil, tablet ve masaüstü uyumlu.
- **Premium Arayüz**: Kırmızı, siyah ve beyazın uyumu.
- **Etkileşimli**: Hover efektleri, scroll animasyonları.
- **Google Maps & İletişim Formu**: Hazır şablonlar.

## Kurulum ve Çalıştırma

Bu bir statik HTML/CSS/JS projesidir. Herhangi bir sunucu kurulumu gerektirmez.

1. `index.html` dosyasını tarayıcınızda açın.

## GitHub Pages ile Yayınlama

1. Bu projeyi GitHub'a yükleyin.
2. Repositories > Settings > Pages menüsüne gidin.
3. **Branch** kısmından `main` (veya `master`) seçin ve kaydedin.
4. Siteniz birkaç dakika içinde yayınlanacaktır.

## Dosya Yapısı
- `assets/images/`: Görseller
- `css/`: Stil dosyaları
- `js/`: JavaScript dosyaları
- `index.html`: Ana sayfa
